# Battleship Game (Console-Based, C++)

## Description

This is a console-based implementation of the classic Battleship game developed as a Programming Fundamentals project. The game allows players to strategically place ships and compete against another player or the computer.

## Features

* Multiplayer mode (Player vs Player)
* Single-player mode (Player vs Computer)
* Option to place ships:

  * Randomly
  * Manually (user-defined positions)
* 6 different ships, each with a specific size
* Turn-based gameplay system
* Limited turns (6 turns per player in Player vs Player mode)
* Alternating turns in Player vs Computer mode
* Hit/Miss detection system

## Concepts Used

* Arrays
* Loops
* Functions
* Conditional statements
* Basic game logic and input handling

##Future Improvements
